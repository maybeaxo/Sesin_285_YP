# WARPOINT VR Games Kiosk

Информационно-справочная система выбора игр виртуальной реальности для парка развлечений WARPOINT.

## Описание

Веб-приложение для сенсорного киоска, позволяющее посетителям парка выбирать VR игры для игры. Приложение оптимизировано для использования на сенсорных экранах в режиме киоска.

## Функциональные возможности

### Для посетителей:
- 📱 Просмотр каталога VR игр с описаниями и скриншотами
- 🎥 Просмотр промо-видео игр
- 🔍 Поиск игр по названию, разработчику или описанию
- 🎮 Фильтрация по жанру, платформе, возрастному ограничению и длительности
- 📊 Просмотр подробной информации об игре (количество игроков, время прохождения, наличие на арене)
- 🎨 Просмотр промо-баннеров на главном экране

### Для администратора:
- 🔐 Защищенная административная панель (пароль: `admin123`)
- ➕ Добавление новых игр
- ✏️ Редактирование существующих игр
- 🗑️ Удаление игр
- 📢 Управление промо-баннерами
- 💾 Автоматическое сохранение изменений в localStorage

## Установка и запуск

### Разработка:
```bash
npm install
npm run dev
```

### Продакшен:
```bash
npm install
npm run build
npm run start
```

### Автоматический запуск на Windows:
1. Создайте ярлык для `start.bat`
2. Добавьте ярлык в автозагрузку Windows
3. Приложение будет автоматически запускаться при включении киоска

## Вход в административную панель

Для входа в административную панель:
1. Удерживайте правый нижний угол экрана в течение 5 секунд
2. Введите пароль: `admin123`

## Технические требования

- **ОС**: Windows 10/11
- **Браузер**: Chrome/Edge (рекомендуется полноэкранный режим)
- **Разрешение**: 1920x1080 (оптимизировано)
- **Управление**: Сенсорный экран

## Настройка для киоска

1. **Полноэкранный режим**: Настройте браузер на запуск в полноэкранном режиме (F11)
2. **Автозапуск**: Добавьте `start.bat` в автозагрузку Windows
3. **Блокировка навигации**: Используйте режим киоска браузера
4. **Отключение заставки**: Настройте Windows на отключение заставки

## Производительность

- Оптимизировано для быстрого отклика (< 2 секунд)
- Использование useMemo для оптимизации фильтрации
- Ленивая загрузка изображений
- Кэширование данных в localStorage
